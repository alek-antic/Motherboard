# Situation 0: Exploring Rules Clarifications
1. Exploration
	1. There are three 8-hour "watches" in a bit. (This can be granulated to six 4-hour watches if everyone prefers)
	2. Each watch, you can march, survey, rest, etc.
2. Resting
	1. If you are in civilization with all needs provided for, normal resting rules apply (short rest is an hour, long rest is a few hours/overnight)
	2. If you are in the wild, a short rest is one "watch". A long rest takes 2 bits without interruption.

1. Mark current watch

	- 2: Signs of a nearby random encounter

	- 5-12: No encounter
3. Declare watch actions
4. Resolve travel
5. Resolve encounters
6. Resolve watch actions
